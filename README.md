# チーム配置シミュレーター

性格特性（MBTI・Big Five）に基づいて新規加入者を既存チームに最適配置するインテリジェントなビジュアルデモアプリケーション

## 🎯 目的

リーダーとメンバーの相性を科学的に分析し、最適なチーム配置を視覚的にわかりやすく提示します。

## ✨ 主な機能

### 🧠 インテリジェントマッチング
- **性格特性分析**: MBTI（16タイプ）とBig Five（5因子モデル）を統合
- **相性スコア計算**: 統計的分析に基づく科学的なマッチングアルゴリズム
- **リアルタイム可視化**: 円形ゲージとカラーバッジでスコアを直感的に表示

### 👥 チーム管理
- **新規加入者の可視化**: 左側に新規加入者を表示
- **既存組織の管理**: 右側に既存チーム（リーダー＋メンバー）を3チーム表示
- **自動配置**: ワンクリックで最適なチームへ自動配置
- **相性詳細表示**: ホバーで相性の理由をツールチップ表示

### 🎨 UX/UI
- **滑らかなアニメーション**: Framer Motionによるプロフェッショナルな動き
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ完全対応
- **アクセシビリティ**: WCAG準拠の高いアクセシビリティ

## 🛠️ 技術スタック

- **Vite**: 高速開発環境
- **React**: UIライブラリ
- **TypeScript**: 型安全性
- **Tailwind CSS**: ユーティリティファーストCSS
- **shadcn/ui**: モダンで洗練されたUIコンポーネントライブラリ
- **Radix UI**: アクセシブルなプリミティブコンポーネント
- **Framer Motion**: 滑らかなアニメーション

## 🧮 マッチングアルゴリズム

### 相性スコア計算式

```typescript
最終スコア = Big Five類似度 + MBTIボーナス
```

#### Big Five類似度（指数関数モデル）
```typescript
similarity = exp(-α × 平均特性差分)
α = 3.0（調整パラメータ）
```

5つの性格因子（開放性、誠実性、外向性、協調性、神経症傾向）の差分を計算し、指数関数で類似度に変換します。

#### MBTIボーナス（最大0.08）
- 同気質（NF/NT/SJ/SP）: +0.04
- E/I一致: +0.02
- J/P一致: +0.02
- N/S一致: +0.015
- T/F一致: +0.015

### 統計的閾値設計

10,000サンプルのシミュレーションに基づく科学的な閾値：

| スコア範囲 | 色 | 意味 | パーセンタイル |
|-----------|---|------|--------------|
| 0.40以上 | 🟢 エメラルド | 非常に良好 | 上位10% |
| 0.30-0.40 | 🔵 ブルー | 良好 | 上位25% |
| 0.20-0.30 | 🟡 アンバー | 普通 | 中央値付近 |
| 0.20未満 | 🟠 オレンジ | 要注意 | 下位25% |

### 配置ロジック

1. 各新規加入者について、全チームリーダーとの相性スコアを計算
2. 最も高い相性スコアのチームに配置
3. 視覚的フィードバック（円形ゲージ＋カラーバッジ）で相性を表示

## 🎨 デザイン仕様

### UIフレームワーク
- **UIキット**: shadcn/ui（2025年最新トレンド対応）
- **カラーシステム**: HSLベースのデザイントークン（ライト/ダークモード対応）
- **アイコン**: Lucide React（モダンで軽量）
- **アニメーション**: Framer Motionによる滑らかなトランジション
- **アクセシビリティ**: Radix UIベースでWCAG準拠

### 相性スコアの可視化

#### 円形プログレスゲージ
- SVGで描画された高品質な円形ゲージ
- `shapeRendering: 'geometricPrecision'`でアンチエイリアシング最適化
- スコアに応じて色が変化（エメラルド/ブルー/アンバー/オレンジ）
- ドロップシャドウエフェクトで視認性向上

#### スコアバッジ
- 控えめで柔らかいデザイン（単色、角丸）
- アバターの絵文字風デザインと調和
- ホバーで詳細ツールチップを表示
- アクセントカラーがスコアと連動

### デザイン原則

1. **統計的根拠**: 10,000サンプルのシミュレーションに基づく閾値設計
2. **視覚的一貫性**: バッジ・ゲージ・ツールチップで統一された色体系
3. **直感的理解**: 色と形状で瞬時に相性を把握
4. **柔らかな印象**: 絵文字風アバターと調和するUI
