# 推奨コマンド

## 開発
```bash
npm run dev
```
Vite開発サーバーを起動します。通常 http://localhost:5173 でアクセスできます。

## ビルド
```bash
npm run build
```
TypeScriptコンパイルとViteビルドを実行し、`dist/`フォルダに本番用ファイルを生成します。

## リンター
```bash
npm run lint
```
ESLintを実行してコード品質をチェックします。

## プレビュー
```bash
npm run preview
```
ビルドされたアプリケーションをローカルでプレビューします。

## システムコマンド（macOS / Darwin）
- `ls` - ファイル一覧
- `cd` - ディレクトリ移動
- `grep` - テキスト検索
- `find` - ファイル検索
- `git` - バージョン管理
