# タスク完了チェックリスト

タスクを完了する前に以下を確認してください：

## 必須チェック
1. **リンター**: `npm run lint` を実行してエラーがないことを確認
2. **ビルド**: `npm run build` を実行してビルドが成功することを確認
3. **動作確認**: `npm run dev` で開発サーバーを起動し、機能が正常に動作することを確認

## コード品質
- TypeScriptの型エラーがないこと
- 未使用のimportや変数がないこと
- console.logなどのデバッグコードが残っていないこと

## Git
- Conventional Commits形式でコミット（feat:、fix:、docs: など）
- 意味のあるコミットメッセージを記述

## ドキュメント
- 必要に応じてREADME.mdを更新
- 複雑なロジックにはコメントを追加
